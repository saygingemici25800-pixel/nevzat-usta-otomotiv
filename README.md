# Nevzat Usta Otomotiv (NUO) — Web Sitesi

Fethiye'de 25+ yıllık tecrübeye sahip oto servis işletmesinin profesyonel web sitesi.

## Özellikler

- Dark theme, mobile-first responsive tasarım
- Chillax Variable font (4 ağırlık sistemi)
- 10+ interaktif animasyon (particle, glowing border, typewriter, gear, marquee)
- SEO optimizasyonu (JSON-LD, Open Graph, semantic HTML)
- WCAG erişilebilirlik uyumluluğu
- 3 hizmet detay sayfası
- Blog sistemi (3 yazı + listeleme)
- İletişim formu + Google Maps

## Hızlı Başlangıç

```bash
# Projeyi klonlayın
git clone <repo-url>
cd nevzat-usta-otomotiv

# Yerel sunucu başlatın
python3 -m http.server 8000

# Tarayıcıda açın
open http://localhost:8000/nevzat-usta-hero2.html
```
